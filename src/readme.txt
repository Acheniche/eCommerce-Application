src - Главная папка.

components содержит компоненты, которые могут быть использованы в разных частях проекта. 
Каждый компонент имеет свою собственную папку с именем компонента и содержит файлы .ts и .css для соответствующего компонента


assets дополнительные компоненты (картинки, стили и тд).


utils содержит утилитарные функции и файлы, которые могут быть использованы в разных частях проект (например работа с api)
